# Soluções Cloud Native

Soluções pós mundo cloud, específicas para rodar em ambiente Cloud, de forma distribuída como clusters.
Exemplo:

- Docker
- Kubernetes, Helm, Istio, Linkerd, Envoy
- Apache Kafka
- Prometheus
- OpenTelemety
- AWS Lambda, Azure Functions etc
- ArgoCD
- Produtos HashiCorp (Valut, Consul, Terraform)

## CNCF (Cloud Native Computing Foundation)

Organização sem fins lucrativos, que cria projetos cloud native e visa fomentar o ambiente Cloud. Patrocina projetos, cria certificações, faz convenções

### Tipos de projeto

#### Graduated

Projetos que já estão consolidados, maduros e testados. Times mais conservadores

### Incubting

Fase média, já possuí uma rodagem, testados. Não tão conservador, mas possuí um grau de certeza

### Sandbox

Projetos promissores, em fase inicial que possuí um grau de risco. Irá possuir um suporte da CNCF
